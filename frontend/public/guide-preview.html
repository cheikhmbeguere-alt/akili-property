🏦 Guide de connexion Pennylane

Suivez ces étapes pour générer et connecter votre token API

⚠️ Important — un token par société
Chaque société (SCI) possède son propre espace Pennylane. Assurez-vous d'être connecté au bon espace avant de générer le token.
1
Accéder à Pennylane
Rendez-vous sur app.pennylane.com et connectez-vous avec votre compte.
💡 Assurez-vous d'être dans le bon espace de travail (la bonne société).
🖼️
Capture d'écran — étape 1
Page de connexion Pennylane
2
Ouvrir les Paramètres
Dans la barre de navigation à gauche, cliquez sur l'icône ⚙️ Paramètres (en bas à gauche de l'écran).
🖼️
Capture d'écran — étape 2
Navigation Pennylane avec icône Paramètres
3
Aller dans la section API
Dans le menu Paramètres, cherchez et cliquez sur l'onglet « API » ou « Intégrations ».
💡 Cette section est réservée aux administrateurs du compte Pennylane.
🖼️
Capture d'écran — étape 3
Menu Paramètres → API
4
Créer une nouvelle clé API
Cliquez sur « Créer une clé API ». Donnez-lui un nom explicite, par exemple : PropManager.
🖼️
Capture d'écran — étape 4
Bouton "Créer une clé API"
5
Sélectionner les permissions
Cochez uniquement la permission « transactions:read ».
💡 Inutile de donner d'autres droits — l'app n'a besoin que de lire vos transactions.
🖼️
Capture d'écran — étape 5
Choix des permissions / scopes
6
Copier le token
Pennylane affiche le token une seule fois. Copiez-le immédiatement.
⚠️ Si vous fermez la fenêtre sans le copier, vous devrez en générer un nouveau.
🖼️
Capture d'écran — étape 6
Token généré à copier
7
Coller dans l'app
Dans PropManager, sélectionnez votre SCI, collez le token et cliquez sur « Connecter Pennylane ».
🖼️
Capture d'écran — étape 7
Champ token dans PropManager
🆘 En cas de problème
  • • Token invalide : vérifiez que vous êtes dans le bon workspace Pennylane
  • • Pas de transactions : vérifiez que le compte bancaire est bien connecté dans Pennylane
  • • Erreur de permission : régénérez le token avec le scope transactions:read
